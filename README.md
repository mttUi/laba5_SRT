# Кастомный мьютекс успешно реализован на основе atomic<bool> и compare_exchange_weak

Программа демонстрирует все три требуемых сценария с подобранными коэффициентами

Синхронизация потоков работает корректно - соблюдается строгая очередность действий

CustomMutex является работоспособной заменой для std::mutex в данной задаче

Использование yield() снижает нагрузку на CPU при ожидании
